# FriendFinder

This is an Express- and Node-based application for matching users based on a personality survey. This application uses Heroku for deployment.

## View Site



Requirements:  Node.js/NPM

## Local Deployment

* Clone repository
* Install modules 
* Run node server.js
* Go to localhost:8000 in web browser

## Technologies Used

* Node.js
* NPM Express
* Heroku
